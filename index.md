# Laser User Manual

## Camera Settings
Ensure that the following settings are enabled on the camera:

Program auto\
Flash off\
ISO-A 1600\
Exposure bias 0\
Single shot\
Underwater shallow white balance\
ESP metering\
Face priority off\
Accessory off\
Jpeg mode underwater\
Autofocus\
Aspect ratio 4000 x 3000\
Output format If + raw\
Still image stabilization on
